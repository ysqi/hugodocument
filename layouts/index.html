无与伦比

Hugo简介，围绕创造网站的又一新思路，Hugo轻松支持多种格式，是创建博客，文档，企业网站，作品展等等的理想工具。 Hugo 神迹般创建网站，给你带来更多乐趣。

跨平台运行

Hugo也许是你安装过最简单的软件，绿色安全，只需下载运行 Hugo不依赖于管理员权限，数据库，运行时环境，语言，外部库。 Hugo创建网站可以发布到S3 S3, GitHub Pages, Dropbox 或者其他任意服务器.

极速 & 强大

Hugo在意编译速度和性能，已采取大量措施已确保快速高效创建网站，我们正在讨论毫秒级创建完整网站的大部分设置。

灵活

Hugo 为你如何使用而进行设计的。 你想要的任何URL格式的内容。 定义属于你自己的内容格式。 基于YAML, TOML or JSON 完全定义属于你自己的元数据。 可检索分类你任何想要的内容。 最棒的是你几乎没有任何配置，Hugo就可以使用起来。
